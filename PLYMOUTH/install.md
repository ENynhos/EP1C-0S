# How to install the EPIC plymouth theme:
## Step One
Download the .zip file in the same folder as this file.
## Step Two
Extract it's contents to the '/usr/share/plymouth/themes' folder.
## Step Three
If you are using KDE Plasma, open 'system settings' -> Boot Splash Screen, and then select the EPIC option, remember to apply changes. \
Otherwise, use the following command:
 sudo plymouth-set-default-theme EPIC
## Step Four
Enjoy your theme
